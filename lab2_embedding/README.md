# Лабораторная работа 2: Embedding с использованием Mistral AI

## Цель

Освоить методы генерации эмбеддингов с помощью Mistral AI, изучить косинусное сходство и визуализировать результаты.

## Задачи

1. Получить API ключ Mistral AI
2. Использовать модель `mistral-embed` для генерации векторных представлений текстов
3. Подготовить набор из 50 предложений и сгенерировать их эмбеддинги
4. Визуализировать результаты в виде heatmap с помощью plotly

## Требования

- **Python 3.12** или выше
- **API ключ Mistral AI**
- **Библиотеки**: `mistralai`, `scikit-learn`, `plotly`, `pandas`

## Установка и настройка

1. **Получение API ключа:**
   - Зарегистрируйтесь на [console.mistral.ai](https://console.mistral.ai/)
   - Создайте API ключ и сохраните в переменную окружения:
   ```bash
   export MISTRAL_API_KEY="your_api_key"
   ```

2. **Установка зависимостей (из корня проекта):**
   ```bash
   uv sync
   ```

3. **Запуск:**
   ```bash
   uv run python lab2_embedding/lab2_embedding.py
   ```
   Или откройте файл в VS Code и запускайте ячейки интерактивно.

## Оценка результатов

- Среднее значение косинусного сходства для схожих пар предложений **должно быть ≥ 0.8**
- Если значение ниже, попробуйте:
  - Подобрать более релевантные пары предложений
  - Использовать предложения из одной предметной области
