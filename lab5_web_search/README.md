# Лабораторная работа 5: LangChain — создание AI-приложений на основе LLM

## Цель

Освоить фреймворк LangChain для создания приложений на основе языковых моделей: цепочки, инструменты и агенты.

## Задачи

1. Изучить архитектуру LangChain и унифицированный интерфейс моделей
2. Реализовать пользовательские инструменты (Tools) для расширения возможностей LLM
3. Создать AI-агента, способного выбирать и использовать инструменты
4. Сравнить автономный и контролируемый подходы к работе с инструментами
5. Сохранить и проанализировать результаты экспериментов

## Требования

- Python 3.12 или выше
- API ключи: `MISTRAL_API_KEY`, `TAVILY_API_KEY`
- Библиотеки: `langchain`, `langchain-mistralai`, `langchain-tavily`, `pandas`

## Установка (из корня проекта)

```bash
uv sync
```

## Запуск

```bash
uv run python lab5_web_search/lab5_web_search.py
```

## Содержание

1. **Что такое LangChain** — обзор фреймворка
2. **Единый интерфейс моделей** — сравнение параметров генерации
3. **Tools в LLM** — калькулятор через прямой вызов и через модель
4. **AI-агенты** — агент с инструментами (погода, население, веб-поиск)
5. **Разделение ответственности** — контролируемый подход с Tavily Search
6. **Анализ данных** — сохранение результатов в CSV и анализ через LLM
