# Лабораторная работа 6: Память и управление состоянием в диалоговых системах

## Цель

Изучить механизмы памяти в диалоговых системах: стратегии управления историей и персистентность состояния.

## Задачи

1. Реализовать базовую память через список сообщений
2. Сравнить стратегии управления историей: полный буфер, окно, суммаризация
3. Построить чат-бот на LangGraph с управлением состоянием
4. Реализовать checkpointing для сохранения состояния между сессиями

## Требования

- Python 3.12 или выше
- API ключ: `MISTRAL_API_KEY`
- Библиотеки: `langchain-mistralai`, `langgraph`

## Установка (из корня проекта)

```bash
uv sync
```

## Запуск

```bash
uv run python lab6_memory_history/lab6_memory_history.py
```

## Содержание

1. **Основы памяти** — ручная передача истории сообщений в LLM
2. **Стратегии управления историей** — сравнение буфера, окна (k=2) и суммаризации
3. **LangGraph чат-бот** — StateGraph + MessagesState
4. **Checkpointing** — MemorySaver, изоляция состояний по thread_id
