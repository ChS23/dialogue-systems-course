# Лабораторная работа 7: Поисковый агент по документам

## Цель

Создать поискового агента на основе LangGraph, способного искать информацию в векторной базе данных и отвечать на вопросы пользователя.

## Задачи

1. Подключиться к векторной БД LanceDB (из lab4)
2. Создать поискового агента на LangGraph с инструментом семантического поиска
3. Реализовать персистентность состояния через checkpointing
4. Обеспечить изоляцию состояний между пользователями

## Требования

- Python 3.12 или выше
- API ключ: `MISTRAL_API_KEY`
- Библиотеки: `langchain-mistralai`, `langgraph`, `lancedb`
- Выполненная lab4 (наличие векторной БД)

## Установка (из корня проекта)

```bash
uv sync
```

## Запуск

```bash
uv run python lab7_document_search/lab7_document_search.py
```

## Содержание

1. **Подключение к БД** — загрузка LanceDB из lab4
2. **Поисковый агент** — LangGraph агент с инструментом поиска по документам
3. **Персистентность** — MemorySaver, сохранение истории диалога
4. **Многопользовательский режим** — изоляция состояний по thread_id
